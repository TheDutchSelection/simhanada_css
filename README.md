# SimhanadaCss

This is Simhanada CSS, the work of CSS mastermind Martijn van Leeuwen.

## Installation

Add this line to your application's Gemfile:

```
gem 'simhanada_css', github: 'TheDutchSelection/simhanada_css', branch: 'master'
```

And then execute:

```
$ bundle
```

## Usage

In your scss file:

```
@import 'simhanada';
```
